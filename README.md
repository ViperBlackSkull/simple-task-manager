# Simple Task Manager

A simple task management application built with Python and Flask.

## Overview

This project provides a web-based task management interface for creating, tracking, and completing tasks.

## Features

- Create and manage tasks
- Track task status (todo, in-progress, done)
- Simple web interface

## Development Setup

Run the init script to set up the development environment:

```bash
./init.sh
```

This will install dependencies and start the development server.
